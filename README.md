# atm_simulation
ATM Simulator is a Python mini project that simulates basic ATM operations such as PIN verification, balance enquiry, withdrawal, and deposit. It uses loops, conditionals, and user input handling. This project helped me practice control flow, debugging, and menu-driven programs.
